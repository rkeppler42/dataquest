# 📱 Profitable App Profiles for the App Store and Google Play Markets

A data analysis project exploring which types of free, English-language mobile apps tend to attract the largest audiences — and are therefore most likely to be profitable through in-app advertising.

This is a **Guided Project** completed as part of the [Data Scientist in Python](https://www.dataquest.io/path/data-scientist/) path on Dataquest.

---

## 🔍 Overview

The goal of this project is to help a hypothetical mobile app company identify profitable app profiles for both the **Apple App Store** and **Google Play Store**. Since the company's revenue model relies on in-app ads, success is measured by the size of the user base an app attracts.

The analysis covers:
- Data cleaning (removing duplicate entries, non-English apps, and paid apps)
- Exploratory analysis of app genres and categories across both markets
- Identifying app profiles that show strong potential in both stores
