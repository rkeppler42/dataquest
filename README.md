# 📊 Dataquest – Data Scientist in Python

This repository contains all the projects I've completed as part of the [Data Scientist in Python](https://www.dataquest.io/path/data-scientist/) path on Dataquest.

> **Note:** Dataquest projects come in two forms:
> - **Guided Projects** – structured projects with step-by-step instructions, completed as part of the curriculum.
> - **Portfolio Projects** – independent projects built to demonstrate skills beyond the guided curriculum.
>
> 📌 Each project folder contains a Jupyter notebook (`.ipynb`) with my full analysis and findings.
